```typescript
import { defineCollection, z } from 'astro:content';

const projects = defineCollection({
  type: 'content',
  schema: z.object({
    title:       z.string(),
    description: z.string(),
    cover:       z.string(),
    images:      z.array(z.string()),
    order:       z.number(),
    liveUrl:     z.string().url().optional(),
    featured:    z.boolean().default(false),
  }),
});

const about = defineCollection({
  type: 'content',
  schema: z.object({}),
});

export const collections = { projects, about };
```

Fields: `title` (string, required), `description` (string, required), `cover` (string, required — path to cover image), `images` (string[], required — array of image paths), `order` (number, required — lower = first on home page), `liveUrl` (string URL, optional), `featured` (boolean, default false — deferred to Phase 2). About collection has empty schema — body is the content.

```css
/* src/styles/global.css */
@import './tokens.css';

  font-family: 'Neue Haas Unica';
  src: url('/fonts/NeueHaasUnica-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'Neue Haas Unica';
  src: url('/fonts/NeueHaasUnica-Medium.woff2') format('woff2');
  font-weight: 500;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'Neue Haas Unica';
  src: url('/fonts/NeueHaasUnica-Italic.woff2') format('woff2');
  font-weight: 400;
  font-style: italic;
  font-display: swap;
}

/* Reset */
*,
*::before,
*::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

/* Base */
html {
  font-family: var(--font-family-base);
  font-size: var(--font-size-base);
  font-weight: var(--font-weight-regular);
  line-height: var(--line-height-body);
  color: var(--color-text);

  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

body {
  min-height: 100vh;
}

img {
  display: block;
  max-width: 100%;
}

a {
  color: inherit;
}

ul {
  list-style: none;
}
```

Phase 1 declares only three font weights: Regular (400), Medium (500), Italic (400i). Bold (700) is added in Phase 2.
Do NOT add an `@import url(...)` for the font file directly — only use `@font-face` declarations. The erroneous `@import url('/fonts/...')` line in the PATTERNS.md global.css example is a mistake — use only `@font-face` blocks.
Font paths start with `/fonts/` which maps to `public/fonts/` at build time.
`font-display: swap` on every `@font-face` to prevent invisible text during load.
