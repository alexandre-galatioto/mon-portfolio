<task type="auto">
  <name>Task 1: Create netlify.toml with build config and cache headers</name>
  <files>netlify.toml</files>
  <read_first>astro.config.mjs, package.json</read_first>
  <action>
Create a new file `netlify.toml` at the project root (same directory as `astro.config.mjs` and `package.json`).

Full file content:

```toml
[build]
  command = "npm run build"
  publish = "dist/"

[[headers]]
  for = "/fonts/*"
  [headers.values]
    Cache-Control = "public, max-age=31536000, immutable"

[[headers]]
  for = "/*.html"
  [headers.values]
    Cache-Control = "public, max-age=0, must-revalidate"
```

Details:
- `command = "npm run build"` runs the Astro build which outputs static HTML/CSS to `dist/`
- `publish = "dist/"` tells Netlify where to find the built site
- Font cache header: 1-year immutable cache because font filenames are content-stable (NeueHaasUnica-Regular.woff2 etc.)
- HTML cache header: must-revalidate so each deploy immediately serves fresh content
- Do NOT add `@astrojs/netlify` adapter — it is not needed for static output (per RESEARCH.md)
- Do NOT add `[context]` or environment blocks — single environment, static site
- Do NOT modify `astro.config.mjs` — `output: 'static'` is already correct
  </action>
